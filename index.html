NCAA Division 1 Football Record Predictor

This project is an NCAA Division I Football Record Predictor, designed to forecast a team’s seasonal performance using a regression model. The model predicts the number of wins and losses based on various factors, including a team’s previous record as well as key offensive and defensive statistics. To evaluate the model’s effectiveness, I tested it using data from the 2019–2020 University of Iowa football team. Based on their statistical profile, the model predicted an 11–2 season; the team ultimately finished 10–3, demonstrating a strong level of accuracy. The project was developed in Excel using data sourced from cfbstats.com.

Price Elasticity Regression Model

As part of a group project for my Business Analytics Capstone course, we partnered with a sponsor company and were provided access to their Amazon sales data. Our objective was to develop a regression model that could estimate both unit sales and price elasticity. Using Python, we built a dynamic model that accepts user input for various product and pricing statistics, allowing it to predict expected unit sales and quantify how percentage changes in price would impact demand.

NCAA College Basketball Database

Developed a comprehensive NCAA Men’s College Basketball database using Oracle APEX, with SQL for querying and data manipulation. The database includes separate tables for each season from 2010 to 2023, containing detailed offensive and defensive statistics for every team from both the regular season and the NCAA tournament. Project data was sourced from Kaggle and organized to support performance analysis and historical comparisons.
